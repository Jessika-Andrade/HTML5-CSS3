Exemplos de semâtica

Negrito

Sem semântica, usamos B.

Com semântica, usamos STRONG

Itálico

Sem semântica, usamos I.

Com semântica, usamos EM.

Marca Texto

Conseguimos fazer uma marcação, utilizando a tag MARK.

Letras Grandes e Pequenas

Para usar letras grandes, BIG.

Para usar letrar pequenas, small.

Texto Deletado

Para usar um texto deletado, só escrever DELL.

Texto Inserido

Para uso semântico, utiliza-se INS.

Para texto sem semântica, usa-se U

Texto Sobescrito

Para escrever acima da palavra, usa-se Sup como no exemplo: 32

Texto Subescrito

Para escrever abaixo da palavra, usa-se SUB como no exemplo: H2O.
